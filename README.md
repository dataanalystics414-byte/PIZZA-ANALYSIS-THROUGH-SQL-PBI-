# 🍕 Pizza Sales Analysis using SQL

## 📌 Overview
This project analyzes pizza sales data using SQL to uncover key business insights such as revenue trends, customer ordering patterns, best-selling pizzas, and category performance. It demonstrates the use of SQL for solving real-world business problems through data analysis.

## 🎯 Objectives
- Analyze total orders and revenue
- Identify top-selling and highest-priced pizzas
- Evaluate sales by category and size
- Analyze hourly and daily order trends
- Calculate revenue contribution and cumulative revenue
- Determine top-performing pizzas by revenue

## 🛠️ Tools Used
- PostgreSQL
- pgAdmin 4
- SQL
- Git & GitHub

## 📂 Dataset
The analysis is based on four tables:
- `orders`
- `order_details`
- `pizzas`
- `pizza_types`

## 📊 SQL Concepts
- Joins
- Aggregate Functions
- GROUP BY
- CTEs
- Window Functions
- Ranking Functions

## 🚀 Key Insights
- Identified top revenue-generating pizzas
- Analyzed customer ordering behavior
- Measured category-wise sales performance
- Tracked revenue trends over time
